# CastKi

Simple app used with Shadowcast3 in lieu of using the web-based page by Genki. Works 100% local, no asking for microphone perms.

## Features

- Live preview from your Shadowcast3 capture card
- Audio passthrough with volume control
- Screenshot and video recording (saved to `~/Videos/Castki`)
- Virtual camera output via OBS Virtual Camera
- Fullscreen mode
- Device switcher if you have multiple capture cards

## Requirements

- Windows 10/11
- [OBS Virtual Camera](https://obsproject.com/) installed (for virtual cam feature)
- Shadowcast3 connected via USB

## Usage

Download `CastKi.exe` from the [Releases](../../releases) page and run it — no install needed.

### Keyboard shortcuts

| Key | Action |
|-----|--------|
| Click video | Toggle control bar |
| `S` | Screenshot |
| `R` | Start / stop recording |
| `F` / `F11` | Fullscreen |
| `Esc` | Exit fullscreen |
